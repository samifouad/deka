This is the practical syntax baseline for writing PHPX daily.

## Variables, functions, return types

```phpx
function add($a: int, $b: int): int {
  return $a + $b
}

echo add(2, 3)
```

Expected output:

```text
5
```

## Interfaces for contracts

```phpx
interface NameProps {
  $name: string
}

function label({ $name }: NameProps): string {
  return "name=" . $name
}
```

## Structs for modeled values

```phpx
struct Package {
  $name: string
  $downloads: int = 0
}

$p = Package { name: "router", downloads: 12 }
echo $p.name . ":" . $p.downloads
```

Expected output:

```text
router:12
```

## JSX in frontmatter pages

```phpx
---
interface HeroProps {
  $title: string
}
function Hero({ $title }: HeroProps) {
  return <h1>{$title}</h1>
}
---
<Hero title="Hello PHPX" />
```

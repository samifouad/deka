Devuelve una representación en forma de cadena de un valor PHP.

Utiliza este helper cuando el tiempo de ejecución necesite una forma de cadena segura para registros o mensajes de error sin mutar el valor en sí.

```php
$value = get_str_func($value);
```

## Ejemplos

```php
<?php

$value = ["status" => "ok", "count" => 3];
$stringValue = get_str_func($value);

echo $stringValue;
```

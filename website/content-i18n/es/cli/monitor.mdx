# deka monitor

Gestionar las comprobaciones de actualización automática (se ejecuta cada 5 minutos).

## Comenzar a monitorear

```bash
deka monitor start
```

## Opciones

- `--upgrade` - Usar actualización en lugar de actualización (reinicia los contenedores)

## Detener monitoreo

```bash
deka monitor stop
```

## Estado

```bash
deka monitor status
```

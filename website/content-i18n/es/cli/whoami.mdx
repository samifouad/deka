# deka whoami

Muestra el usuario y rol actuales en esta instancia.

```bash
deka whoami
```

## Opciones

- `-s, --server <url>` - URL de la API de Deka (predeterminado: http://localhost:8520)

## Ejemplo de salida

```text
$ deka whoami
@samifouad (admin)
```

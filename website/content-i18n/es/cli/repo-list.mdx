# lista de repositorios de deka

Lista los repositorios en el servidor de deka.

```bash
deka repo list
```

Alias: `deka repo ls`

## Opciones

- `-q, --jq <expression>` - Filtrar la salida JSON usando una expresión jq
- `--json [fields]` - Salida JSON con los campos especificados
- `-l, --language <lang>` - Filtrar por lenguaje de programación principal
- `--topic <topics...>` - Filtrar por tema

## Ejemplo de salida

```text
$ deka repo list
core/runtime
core/identity
apps/deka-dashboard
```

# deka c run

Crea e inicia un contenedor a trav√©s de deka-agent.

```bash
deka c run <name> <image>
```

## Opciones

- `-p, --port <mapping...>` - Mapeos de puertos (host:contenedor)
- `-e, --env <var...>` - Variables de entorno (CLAVE=VALOR)
- `-d, --detach` - Ejecutar en segundo plano (predeterminado: verdadero)

Ejemplo:

```bash
deka c run web nginx:latest -p 8080:80 -e NODE_ENV=production
```

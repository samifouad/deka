# registros de deka

Seguir los registros de un servicio específico.

```bash
deka logs [service]
```

## Opciones

- `-f, --follow` - Seguir la salida de registros
- `-n, --tail <lines>` - Número de líneas a mostrar (predeterminado: 100)

Sigue los registros en tiempo real:

```bash
deka logs -f deploy
```

## Ejemplo de salida

```text
$ deka logs -f deploy
[deploy] starting build 7f9b2c
[deploy] cloning repo deka-example
[deploy] build complete in 42s
```

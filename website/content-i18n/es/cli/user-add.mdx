# deka user add

Agrega un usuario por nombre de usuario o clave pública.

```bash
deka user add [username]
```

## Opciones

- `--pubkey <key>` - Agregar usuario por clave pública directamente
- `--role <role>` - Rol: administrador o miembro (predeterminado: miembro)
- `-s, --server <url>` - URL de la API de Deka

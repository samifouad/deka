# actualización de deka

Obtén las últimas imágenes sin reiniciar los contenedores.

```bash
deka update
```

## Opciones

- `-a, --all` - Obtener todas las imágenes, no solo las desactualizadas
- `-q, --quiet` - Suprimir la salida de progreso

Verifica si hay actualizaciones primero:

```bash
deka check
```

## Ejemplo de salida

```text
$ deka update
pulling ghcr.io/tananetwork/edge:1.12.0
pull complete
```

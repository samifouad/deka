# deka run

Ejecuta un punto de entrada de JavaScript o TypeScript con deka-runtime.

```bash
deka run <entry> [args...]
```

## Opciones

- `--deka` - Fuerza deka-runtime incluso si la entrada tiene un shebang de node/bun
- `--runtime-bin <path>` - Ruta al binario de deka-runtime

Ejemplo:

```bash
deka run ./scripts/hello.ts
```

Fuerza deka-runtime para una CLI de Node/Bun:

```bash
deka run --deka vite
```

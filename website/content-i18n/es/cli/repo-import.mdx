# importación de repositorio de deka

Importa un repositorio de GitHub con problemas y solicitudes de extracción.

```bash
deka repo import <source>
```

## Opciones

- `-s, --server <url>` - URL del servidor Deka (predeterminado: http://localhost:9090)

# deka start

Inicia toda la pila de hosting de Deka.

```bash
deka start
```

## Opciones

- `--genesis` - Inicializa el bloque génesis en el primer inicio
- `-d, --detach` - Ejecutar en segundo plano (predeterminado: verdadero)
- `--infra` - Iniciar solo servicios de infraestructura (postgres, redis) y exponer puertos para desarrollo local

El modo de infraestructura expone:

- postgres: `postgres://tana:tana_dev_password@localhost:5432/tana`
- redis: `redis://localhost:6380`
- pgbouncer: `postgres://tana:tana_contract_password@localhost:6432/<db>`

## Ejemplo de salida

```text
$ deka start
✓ docker running
✓ postgres ready
✓ redis ready
✓ migrations complete

✓ t4 ready
```

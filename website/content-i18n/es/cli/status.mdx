# estado de deka

Muestra la salud y el estado de ejecución de cada servicio.

```bash
deka status
```

Ejemplo de salida:

```text
● [postgres] healthy :5432
● [redis] healthy :6379
● [edge] running :8506
● [t4] running :8507
● [deka-git] running :8508
● [deploy] running :8509
```

## Ejemplo de salida

```text
$ deka status
● [postgres] healthy :5432
● [redis] healthy :6379
● [edge] running :8506
● [t4] running :8507
● [deka-git] running :8508
● [deploy] running :8509
```

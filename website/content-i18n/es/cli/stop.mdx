# deka stop

Detener los servicios de Deka o servicios específicos por nombre.

```bash
deka stop <services...>
```

## Opciones

- `--volumes` - Eliminar volúmenes (los datos se perderán)

Para detener todo y eliminar volúmenes:

```bash
deka stop all --volumes
```

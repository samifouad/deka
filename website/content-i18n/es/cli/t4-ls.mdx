# deka t4 ls

Lista de objetos en un bucket T4.

```bash
deka t4 ls <bucket>
```

## Opciones

- `--prefix <prefix>` - Filtrar por prefijo
- `--json` - Salida en JSON

## Ejemplo de salida

```text
$ deka t4 ls assets --prefix logos/
logos/deka.svg
logos/deka-dark.svg
```

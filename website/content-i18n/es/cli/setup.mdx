# configuración de deka

Ejecuta la configuración interactiva antes de tu primer despliegue. Esto escribe la configuración y prepara la autenticación de tu registro.

```bash
deka setup
```

## Lo que configura

- Acceso al registro de contenedores
- Valores predeterminados del servicio en tiempo de ejecución
- Preferencias de despliegue y rutas de almacenamiento

La configuración se almacena en `~/.config/deka/`, por lo que puedes volver a ejecutar la configuración de forma segura cuando necesites rotar las credenciales.

Si la autenticación del registro falla, también puedes ejecutar:

```bash
docker login ghcr.io
```

# deka c ps

Lista de contenedores gestionados por el agente de deka.

```bash
deka c ps
```

## Opciones

- `-a, --all` - Mostrar todos los contenedores (incluidos los detenidos)
- `--json` - Salida en JSON

## Ejemplo de salida

```text
$ deka c ps
NAME        IMAGE                          STATUS
edge        ghcr.io/tananetwork/edge:1.12  running
postgres    postgres:16                    running
```

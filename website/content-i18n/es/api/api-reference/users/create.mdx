---
title: /usuarios
description: Crea un nuevo usuario en la blockchain.
---
Crea una nueva cuenta de usuario enviando una transacción de creación de usuario. La transacción estará pendiente hasta que se incluya en un bloque.

<details>
 <summary><code className="text-green-400">POST</code> <code><b>/users</b></code> <code>(crear usuario)</code></summary>

### Solicitud

```bash
curl -X POST http://localhost:8080/users \
  -H "Content-Type: application/json" \
  -d '{
    "publicKey": "ed25519_abc123...",
    "username": "@alice",
    "displayName": "Alice Johnson",
    "bio": "Software developer"
  }'
```

### Parámetros

<table>
 <tr>
    <td>_parámetro_</td>
    <td>_tipo_</td>
    <td>_necesidad_</td>
  </tr>
 <tr>
    <td>**publicKey**</td>
    <td>cadena</td>
    <td>requerido</td>
  </tr>
 <tr>
    <td>**username**</td>
    <td>cadena</td>
    <td>requerido</td>
  </tr>
 <tr>
    <td>**displayName**</td>
    <td>cadena</td>
    <td>opcional</td>
  </tr>
 <tr>
    <td>**bio**</td>
    <td>cadena</td>
    <td>opcional</td>
  </tr>
</table>

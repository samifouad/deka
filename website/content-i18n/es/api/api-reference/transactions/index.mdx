---
title: /transactions
description: Gestionar transacciones de blockchain
---
Crea y consulta transacciones en la blockchain. Las transacciones permanecen en un estado pendiente hasta que se incluyen en un bloque.

<details>
 <summary><code className="text-blue-400">GET</code> <code><b>/transactions</b></code> <code>(obtener todas las transacciones)</code></summary>

### Solicitud

```bash
curl "http://localhost:8080/transactions?limit=50&offset=0"
```

### Parámetros

<table>
 <tr>
    <td>_parámetro_</td>
    <td>_tipo_</td>
    <td>_necesidad_</td>
  </tr>
 <tr>
    <td>**limit**</td>
    <td>número</td>
    <td>opcional (por defecto: 100)</td>
  </tr>
 <tr>
    <td>**offset**</td>
    <td>número</td>
    <td>opcional (por defecto: 0)</td>
  </tr>
</table>

### Respuesta

<table>
 <tr>
    <td>código http</td>
    <td>tipo de contenido</td>
    <td>cuerpo</td>
  </tr>

    <td>application/json</td>
<td>

<details>
 <summary><code className="text-green-400">POST</code> <code><b>/transactions</b></code> <code>(crear transacción)</code></summary>

### Solicitud

```bash
curl -X POST http://localhost:8080/transactions \
  -H "Content-Type: application/json" \
  -d '{
    "from": "usr_abc123",
    "to": "usr_def456",
    "amount": "100.00",
    "currencyCode": "USD",
    "type": "transfer",
    "signature": "sig_..."
  }'
```

### Parámetros

<table>
 <tr>
    <td>_parámetro_</td>
    <td>_tipo_</td>
    <td>_necesidad_</td>
  </tr>
 <tr>
    <td>**from**</td>
    <td>cadena (UUID)</td>
    <td>requerido</td>
  </tr>
 <tr>
    <td>**to**</td>
    <td>cadena (UUID)</td>
    <td>requerido</td>
  </tr>
 <tr>
    <td>**amount**</td>
    <td>cadena (decimal)</td>
    <td>opcional*</td>
  </tr>
 <tr>
    <td>**currencyCode**</td>
    <td>cadena</td>
    <td>opcional*</td>
  </tr>
 <tr>
    <td>**type**</td>
    <td>cadena</td>
    <td>requerido</td>
  </tr>
 <tr>
    <td>**signature**</td>
    <td>cadena</td>
    <td>requerido</td>
  </tr>
 <tr>
    <td>**contractId**</td>
    <td>cadena (UUID)</td>
    <td>opcional</td>
  </tr>
 <tr>
    <td>**contractInput**</td>
    <td>objeto</td>
    <td>opcional</td>
  </tr>
</table>

**Tipos de transacción:** `transfer`, `deposit`, `withdraw`, `contract_call`, `user_creation`, `contract_deployment`

*Requerido para tipos de transferencia/deposito/retiro

    <td>201</td>
    <td>application/json</td>
<td>

```json

  "id": "tx_1a2b3c4d5e6f",

  "to": "usr_def456",
  "amount": "100.00",
  "currencyCode": "USD",
  "type": "transfer",
  "status": "pending",
```

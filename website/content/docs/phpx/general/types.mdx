PHPX emphasizes explicit contracts at component and function boundaries.

## Interface contract + destructuring

```phpx
interface UserProps {
  $name: string
  $age: int
}

function UserLabel({ $name, $age }: UserProps): string {
  return $name . " (" . $age . ")"
}
```

## Struct as data model

```phpx
struct User {
  $name: string
  $age: int
}

$u = User { name: "Ava", age: 30 }
echo $u.name
```

Expected output:

```text
Ava
```

## Optional values with Option

```phpx
function printEmail($email: Option<string>): string {
  return match ($email) {
    Some($value) => $value
    None => "no-email"
  }
}
```

## Fallible operations with Result

```phpx
function parsePort($value: string): Result<int, string> {
  if (!isNumeric($value)) {
    return Err("invalid port")
  }
  return Ok(toInt($value))
}
```

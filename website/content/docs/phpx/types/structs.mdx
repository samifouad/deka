# PHPX Types and Structs

PHPX provides a robust type system with compile-time type checking and inference.

## Basic Types

```phpx
$number: int = 42;
$price: float = 19.99;
$name: string = "PHPX";
$active: bool = true;
```

## Structs

Structs are the primary way to define custom types in PHPX:

```phpx
struct User {
    $name: string;
    $age: int = 0;  // Default value
    $email: string;
}

$user = User {
    $name: "Alice",
    $email: "alice@example.com"
};
```

## Option and Result Types

PHPX uses Option and Result types for safe error handling:

```phpx
function findUser($id: int): Option<User> {
    // Returns Some($user) or None
}

function divide($a: int, $b: int): Result<int, string> {
    if ($b === 0) {
        return Result::Err("Division by zero");
    }
    return Result::Ok($a / $b);
}
```

## Type Inference

PHPX can often infer types:

```phpx
$users = [User { $name: "Alice" }, User { $name: "Bob" }];
// $users is inferred as array<User>
```

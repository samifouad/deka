    # deka update

Pull the latest images without restarting containers.

```bash
deka update
```

## Options

- `-a, --all` - Pull all images, not just outdated ones
- `-q, --quiet` - Suppress progress output

Check for updates first:

```bash
deka check
```

## Example output

```text
$ deka update
pulling ghcr.io/tananetwork/edge:1.12.0
pull complete
```

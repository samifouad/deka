# deka status

Show the health and runtime status of each service.

```bash
deka status
```

Example output:

```text
● [postgres] healthy :5432
● [redis] healthy :6379
● [edge] running :8506
● [t4] running :8507
● [deka-git] running :8508
● [deploy] running :8509
```

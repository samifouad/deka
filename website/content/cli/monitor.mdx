    # deka monitor

Manage automatic update checks (runs every 5 minutes).

## Start monitoring

```bash
deka monitor start
```

## Options

- `--upgrade` - Use upgrade instead of update (restarts containers)

## Stop monitoring

```bash
deka monitor stop
```

## Status

```bash
deka monitor status
```

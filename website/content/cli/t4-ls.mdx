# deka t4 ls

List objects in a T4 bucket.

```bash
deka t4 ls <bucket>
```

## Options

- `--prefix <prefix>` - Filter by prefix
- `--json` - Output JSON

## Example output

```text
$ deka t4 ls assets --prefix logos/
logos/deka.svg
logos/deka-dark.svg
```

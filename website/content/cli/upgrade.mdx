# deka upgrade

Stop, pull, restart outdated containers and cleanup old images.

```bash
deka upgrade
```

## Options

- `-q, --quiet` - Suppress progress output

## Example output

```text
$ deka upgrade
stopping edge
pulling ghcr.io/tananetwork/edge:1.12.0
starting edge
cleaned up old images
```

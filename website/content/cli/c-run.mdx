# deka c run

Create and start a container via deka-agent.

```bash
deka c run <name> <image>
```

## Options

- `-p, --port <mapping...>` - Port mappings (host:container)
- `-e, --env <var...>` - Environment variables (KEY=VALUE)
- `-d, --detach` - Run in background (default: true)

Example:

```bash
deka c run web nginx:latest -p 8080:80 -e NODE_ENV=production
```

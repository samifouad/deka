PHP values fall into a handful of core types. This section documents how each
one behaves inside the Deka runtime and where to look for PHP reference details.

## Type reference

- [string](/docs/php/types/string) - Text values and encoding behavior.
- [int](/docs/php/types/int) - Integer values and limits.
- [float](/docs/php/types/float) - Floating-point numbers.
- [bool](/docs/php/types/bool) - Boolean values.
- [array](/docs/php/types/array) - Ordered maps and list-like arrays.
- [object](/docs/php/types/object) - Class instances and stdClass objects.
- [callable](/docs/php/types/callable) - Values that can be invoked.
- [null](/docs/php/types/null) - Absence of a value.
- [mixed](/docs/php/types/mixed) - Any possible value.

## Notes

These pages are based on the PHP manual and highlight any runtime-specific
behavior in Deka.

    # deka repo list

List repositories on the deka server.

```bash
deka repo list
```

Aliases: `deka repo ls`

## Options

- `-q, --jq <expression>` - Filter JSON output using a jq expression
- `--json [fields]` - Output JSON with the specified fields
- `-l, --language <lang>` - Filter by primary coding language
- `--topic <topics...>` - Filter by topic

## Example output

```text
$ deka repo list
core/runtime
core/identity
apps/deka-dashboard
```

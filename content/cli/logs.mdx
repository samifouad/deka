    # deka logs

Tail logs for a specific service.

```bash
deka logs [service]
```

## Options

- `-f, --follow` - Follow log output
- `-n, --tail <lines>` - Number of lines to show (default: 100)

Follow logs in real time:

```bash
deka logs -f deploy
```

## Example output

```text
$ deka logs -f deploy
[deploy] starting build 7f9b2c
[deploy] cloning repo deka-example
[deploy] build complete in 42s
```

    # deka c ps

List containers managed by the deka-agent.

```bash
deka c ps
```

## Options

- `-a, --all` - Show all containers (including stopped)
- `--json` - Output JSON

## Example output

```text
$ deka c ps
NAME        IMAGE                          STATUS
edge        ghcr.io/tananetwork/edge:1.12  running
postgres    postgres:16                    running
```

    # deka run

Execute a JavaScript or TypeScript entrypoint with deka-runtime.

```bash
deka run <entry> [args...]
```

## Options

- `--deka` - Force deka-runtime even if the entry has a node/bun shebang
- `--runtime-bin <path>` - Path to the deka-runtime binary

Example:

```bash
deka run ./scripts/hello.ts
```

Force deka-runtime for a Node/Bun CLI:

```bash
deka run --deka vite
```
